Run MYSTIC in tilted independent pixel mode. Only meaningful with rte_solver montecarlo. Choose either mc_tipa dir3d or mc_tipa dirdiff. The latter is the analogon to tipa dirdiff, where the 3d-cloud-matrix is tilted according to the solar zenith angle and is then used as input for all the following calculations. In mc_tipa dir3d, photons are propagated in 3d until the first scattering occurs, after that the photon is captured within the current column.
