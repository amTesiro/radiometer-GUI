Set the wavelengths to be selected. To be used together with predefined wavelength grids, such as wavelength_grid_file, mol_tau_file abs file and particularly useful in combination with the mol_abs_param option where often only a specified number of wavelength bands is required. E.g., in combination with mol_abs_param AVHRR_KRATZ, wavelength_index 15 15 will select wavelength index 15 which corresponds to channel 4, or wavelength_index 10 14 will select those bands required for channel 3. Indices start from 1.
