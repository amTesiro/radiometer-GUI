Define a MYSTIC 2D albedo input file (default: albedo2D.dat).  

mc_albedo_file file

 
The format of the albedo file is:  

Nx Ny dx dy ix iy albedo ...

 
where Nx and Ny are the number of grid boxes in x- and y-direction, dx and dy are the size of the pixels in km. In the second and the following lines the indices in x- and y-direction (ix=1…Nx; iy=1…Ny) and the albedo of the pixel are specified.
