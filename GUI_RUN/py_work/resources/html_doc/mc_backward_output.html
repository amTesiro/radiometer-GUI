Specify quantity to be calculated using backward Monte Carlo.  

mc_backward_output output [unit]

 
So far the following output options have been implemented:

edir

direct horizontal irradiance

edn

diffuse downward irradiance (default)

eup

diffuse upward irradiance

act

actinic flux

abs

absorption

emis

emission

heat

heating rates, that is absorption + emission

For abs, emis, heat an optional argument W_per_m2_and_dz (default), W_per_m3, or K_per_day may be specified which converts the result e.g. to heating rates.
