Salinity for cox54a,cox54b ocean BRDF (in ”per mille”, 0.1%; this unit is equivalent to the other common units for salinity, ppt - parts per thousand, psu - practical salinity unit).  

cox_and_munk_sal value

 
At present only available with rte_solver cdisort, mystic_doc\ifthenelse\booleanrte_solver montecarlo and rte_solver fdisort2. The number of streams (nstr) is automatically increased to 16 if cox_and_munk BRDF is switched on, to avoid numerical problems. The default value is 34.3. To switch on Cox and Munk BRDF, specify any of the cox_and_munk options and define at least cox_and_munk_u10.
