Switch on OpenGL visualization for MYSTIC.  

mc_visualize [hiddenline]

 
Use the optional argument hiddenline to switch on hidden line removal for the MYSTIC online visualization. Good for topography, not so good for clouds because the latter look much more realistic when the hidden layers are plotted in transparent mode.

Try the following keys in the graphics window:

a

clouds darker

A

clouds brighter

h

decrease displayed domain height

H

increase displayed domain height

r

turn right

l

turn left

u

turn up

d

turn down

k

”Karussell” (rotate continuously)

i

zoom in

o

zoom out

f

fullscreen

q

quit

w

wireframe (frame cloudy pixels)

s

store and release photon paths; press once to start displaying every 10000th photon pass; press again to save last photon path; press again to release photon

b

step through photon path; store a photon with 's'; press 'b'; use '+' and '-' to step through the photon path forward and backward
