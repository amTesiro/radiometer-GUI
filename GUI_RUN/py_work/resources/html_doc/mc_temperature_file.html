Define a MYSTIC 2D temperature input file.  

mc_temperature_file file

 
The expected format of the temperature file is:  

Nx Ny dx dy ix iy temperature

 
where Nx and Ny are the number of grid boxes in x- and y-direction, dx and dy are the size of the grid boxes in km. In the second and the following lines the indices in x- and y-direction and the temperature of the pixel are specified.
