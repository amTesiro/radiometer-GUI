Switch scattering off.  

no_scattering [name]

 
If name is not set, all scattering (molecular, aerosol, cloud, ice cloud, and any profile) is switched off.

Possible choises for the optional argument name are:

mol

Switch off molecular scattering.

aer

Switch off scattering by aerosols.

wc

Switch off scattering by water clouds.

ic

Switch off scattering by ice clouds.

profile

Switch off scattering by any profile defined in profile typename.
